Portofolio Kami

SundaDigi

Aplikasi edukasi budaya Sunda kolaborasi dengan Universitas Padjadjaran (Unpad). Fitur unggulannya antara lain kamus, audio carpon, aksara Sunda, tanya PR, koleksi karya sastra, hingga pelatihan budaya. Dikembangkan untuk membantu siswa dan masyarakat mempelajari bahasa Sunda secara praktis dan interaktif.

DXPortable

Platform digital untuk komunitas amatir radio yang mencakup sistem aktivasi lokasi (pantai, taman, gunung, dll.), pencatatan log otomatis, fitur QSL card, leaderboard, award, fitur topup & shop, serta ruang chat antar anggota. Sistem ini juga dilengkapi belasan website subdomain pendukung yang terintegrasi untuk event dan kegiatan komunitas.

SmartQuiz

Aplikasi kuis Android edukatif yang bisa dimainkan offline, dengan kategori pelajaran dan umum, sistem level, leaderboard, avatar, dan shop. Cocok untuk pelajar hingga profesional. Tersedia gratis di Play Store sejak 2022, dikembangkan ulang mulai Mei 2025.

CatatAja

Aplikasi pencatatan kas dan utang untuk organisasi atau usaha kecil. Mendukung login, pengelompokan laporan, pencatatan uang keluar masuk per individu, fitur tagih & pelaporan akhir, ekspor ke Excel, serta input data satuan maupun massal (batch). Semua fitur terintegrasi dengan hosting melalui API sehingga bisa digunakan secara online.

Absensi Face Recognition

Aplikasi absensi Android dengan teknologi pengenalan wajah menggunakan ML Kit dan OpenCV. Sistem ini memverifikasi wajah pengguna dengan data server, serta mencocokkan lokasi GPS saat absen dengan titik lokasi yang diizinkan menggunakan perhitungan jarak (geo-distance). Seluruh proses terhubung dengan server via API, mendukung validasi real-time, pelaporan akurat, dan integrasi database untuk keperluan lembaga atau institusi.

Ratusan Proyek & Sistem Kustom

LamSTech telah menyelesaikan ratusan website untuk berbagai kebutuhan seperti profil usaha, komunitas, akademik, serta landing page untuk promosi produk, UMKM, dan portofolio digital dengan desain modern dan responsif. Kami juga membangun sistem ringan seperti katalog, portal akademik sederhana, dashboard komunitas, dan integrasi API sesuai permintaan klien. Di samping itu, tersedia pula puluhan aplikasi Android serta proyek coding khusus seperti web scraping, form dinamis, dan layanan berbasis hosting. Semua dikembangkan secara fleksibel, efisien, dan disesuaikan 100% dengan kebutuhan pengguna.
